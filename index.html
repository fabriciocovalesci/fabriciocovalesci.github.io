$ IOT

Controle de acesso com reconhecimento facial usando arduino.

Este projeto foi desenvolvido na disciplina de Tópicos Especiais em Informática como trabalho final.
Abaixo segue algumas espeficicações:

  • Reconhecimento facial através da webcam do computador
  • Uso de bibliotecas como OpenCV e Face Recognition
  • Módulo relé com dois canais para Arduino

Permite o acionamento de travas magnéticas, iluminação, com, bloqueio, etc.
Ao reconhecer a face do usuário autorizado, aciona uma das lâmpadas (liberação da trava).
Se reconhecer uma face não autorizada, aciona a outra lâmpada (aviso de intruso). Se não reconhecer uma face desativa ambas as lâmpadas.

Acionamento das lâmpadas(poderia ser travas) por meio de réles.

Quando a webcam do notebook idêntifica o usuário cadastrado, aciona um relé, que fecha circuito ligando a lâmpada(ou trava)

Repositório do projeto pode ser encontrado neste link

$ Projetos

Sistema de Gestão de Horários para IFRS - Campus Sertão

Prosposta inicial do projeto era desenvolver um sistema de gestão de horários, com finalidade de substituir o preenchimento manual em planilhas compartilhadas, que causava grande transtorno. Projeto não chegou a ser concluido como um todo, por conta da complexidade e também por ter que aprender as tecnologias que seriam utilizadas durante o projeto.
Tecnologias utilizadas:

  • Framework Pyramid
  • Python
  • Javascript
  • HTML
  • CSS
  • GitLab
  • PostgreSQL
