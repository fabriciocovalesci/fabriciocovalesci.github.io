Sistema de Gestão de Horários para IFRS - Campus Sertão

Prosposta inicial do projeto era desenvolver um sistema de gestão de horários, com finalidade de substituir o preenchimento manual em planilhas compartilhadas, que causava grande transtorno. Projeto não chegou a ser concluido como um todo, por conta da complexidade e também por ter que aprender as tecnologias que seriam utilizadas durante o projeto.
Tecnologias utilizadas:

  • Framework Pyramid
  • Jinja2
  • Python
  • Javascript
  • HTML5
  • CSS3
  • GitLab
  • PostgreSQL

Projeto:​ Vida Segura O protótipo VIDA SEGURA tem o intuito de trazer leads quentes para o corretor poder trabalhá-los e convertê-los no maior número de vendas possível. Por meio de técnicas de SEO, Growth Hacking e Copywriting, o Marketing Digital visa crescer exponencialmente a visualização do site para atrair os usuários oriundos das redes sociais. O Hackathon foi totalmente online, promovido pela Shawee - Hackathons management, juntamente com a empresa Gr1d Insurance. Tecnologias utilizadas para desenvolvimento do protótipo foram: Django, HTML, CSS, JavaScript, jQuery.
